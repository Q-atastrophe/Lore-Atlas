
Hi! Been working on an extension for myself and wanted to share it! It's called Lore Atlas.

Lore Atlas is a visual lore management extension for SillyTavern — Worlds, Lorebooks, Scenes, and Entries.

**What can you do with it?**
---------------------------------
* Organize lorebooks into Worlds (Faelands, Demonworld, Harry Potterverse, Modern AU) — multi-World assignment supported. AUS FOR EVERYONE!
* Create Lorebooks within Lore Atlas (and have access to the same entries as Silly Tavern's native Lorebook entries do) and have them show up in the Silly Tavern native UI and Vice Versa
* Create Scenes — "Harry Potter Year 4," "MHA Demon AU" — that activate specific lorebook subsets with one click; handy for information firewalls and AU swaps
* Attach cover images at every level (Worlds, Lorebooks, Scenes, Entries) via drag-from-desktop or file picker
* Edit entries with auto-save, per-entry undo, and a layout that mirrors SillyTavern's native editor
* Switch contexts from a floating launcher that lives outside the extensions menu (Both worlds and scenes)
* Browse in grid or list view at every level, with tags for filtering

**What it do?**
----------------
Lore Atlas runs as a SillyTavern extension and sits on top of your existing native lorebook JSON files — it never modifies them. All Atlas-specific data (Worlds, Scenes, images, tags) lives in extension settings, so uninstalling leaves your lorebooks completely intact, with cross-functionality which means that if you add something in the Silly Tavern World Info panel, it gets added into Lore Atlas and vice versa!

As an aside, this is all completely vibe coded and was made for my own use because I like having AU's within my worlds and I didn't like the tedium of Silly Tavern's native UI. So if there are issues, please don't scream at me- let me know and I'll try to fix them!

Hope you enjoy!
